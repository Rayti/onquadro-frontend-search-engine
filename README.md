# ONQUADRO Frontend

![workflow](https://github.com/Rayti/onquadro-frontend-search-engine/actions/workflows/full.yml/badge.svg?branch=dev)


## Requirements

- `aspnet-runtime-2.2`
- `dotnet-runtime-2.2`
- `dotnet-sdk2.2`

## Building

```sh
cd RNAqbase
dotnet build
```

## Building Angular application

```sh
cd RNAqbase/ClientApp
ng build --prod
```
